    HTML

    HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web. Dengan HTML, para pengembang web dapat membuat struktur dasar dari sebuah halaman web, termasuk teks, gambar, hyperlink, dan elemen-elemen lainnya. Setiap elemen dalam HTML didefinisikan dengan menggunakan tag, yang dimulai dengan tanda kurung sudut (<) dan diakhiri dengan tanda kurung sudut lainnya (>). Misalnya, tag digunakan untuk mendefinisikan paragraf, sedangkan digunakan untuk menampilkan gambar. Selain itu, HTML juga mendukung atribut yang memberikan informasi tambahan tentang elemen, seperti warna, ukuran, atau lokasi. Dengan HTML, pengembang web dapat menciptakan halaman web yang interaktif dan dinamis, serta mengintegrasikan konten multimedia seperti audio dan video. HTML merupakan bagian fundamental dari pengembangan web dan merupakan dasar dari teknologi web modern.

    CSS

    CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengontrol tampilan dan format dari halaman web yang ditulis dalam HTML. Dengan CSS, pengembang web dapat memisahkan struktur dan konten dari presentasi visual, memungkinkan mereka untuk membuat tampilan yang konsisten dan menarik secara visual untuk situs web mereka. CSS menggunakan aturan yang disebut selektor untuk menargetkan elemen HTML yang ingin dimodifikasi, dan mendefinisikan properti seperti warna, ukuran, font, dan tata letak. Misalnya, dengan menggunakan CSS, pengembang dapat menentukan bahwa semua teks dalam elemen akan memiliki warna tertentu, ukuran font yang konsisten, dan jarak antarbaris yang diinginkan. Selain itu, CSS juga mendukung konsep kaskade dan warisan, yang memungkinkan pengembang untuk menerapkan gaya secara hierarkis, dengan kemampuan untuk mewarisi gaya dari elemen yang lebih besar atau dari gaya yang sudah ditentukan sebelumnya. Dengan CSS, pengembang web dapat menciptakan tampilan yang responsif, fleksibel, dan mudah diubah, memungkinkan pengalaman pengguna yang lebih baik di berbagai perangkat dan platform. CSS merupakan komponen integral dari desain web modern dan merupakan alat yang kuat untuk mengontrol estetika dan fungsionalitas situs web.
